# MAITRI – AI Space Companion

MAITRI is a multimodal emotional AI companion designed to support astronauts during long-duration space missions. The system analyzes visual and vocal cues to detect emotional states and respond empathetically.

The goal is to reduce psychological isolation experienced by astronauts in deep-space environments.

---

# Research Objective

This project focuses on identifying the most efficient combination of offline AI models for multimodal emotion recognition.

Key evaluation factors:

- Processing latency
- CPU/GPU usage
- Model accuracy
- Offline capability

---

# System Architecture

Input Sources:

1. Camera
2. Microphone

Processing Modules:

Visual Emotion Detection
- Face Emotion Recognition
- Body Posture Analysis

Speech Emotion Detection
- Speech to Text
- Sentiment Analysis

Fusion Engine

Final emotional state is generated by combining outputs from visual and speech models.

---

# Prototype Research Combinations

We are currently testing the following model combinations:

1. Vosk + DeepFace
2. Vosk + MediaPipe
3. Whisper + DeepFace

Each combination is evaluated for:

- Response latency
- Processing efficiency
- Emotional recognition performance

# Project Pipeline

Camera Input
→ Face Detection
→ Facial Emotion Analysis

Camera Input
→ Body Pose Detection
→ Posture Emotion Analysis

Microphone Input
→ Speech to Text
→ Sentiment Detection

Fusion Layer
→ Combine Visual + Speech Emotions

Emotion Response Engine
→ AI Companion Interaction

# Offline Capability

All models are designed to run in offline mode to support space missions where internet connectivity may be unavailable.
